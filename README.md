# Constance Autosplitter (WIP)
## Features
Autostart, Autosplit (only at the end for now), and uses Game Time.
## How to add it to Livesplit
1. Download [asl-help](https://github.com/just-ero/asl-help/raw/refs/heads/main/lib/asl-help) then navigate to your Livesplit installation folder and put the file in the **Components** folder.
2. Download **Constance.asl** from this repository (Click on Constance.asl, then **Download raw file**) <img width="150" height="50" alt="Showing where the Download raw file icon is" src="https://github.com/user-attachments/assets/87d2f51d-5510-4897-b45b-ac0250e390ef" />
3. Launch Livesplit.exe then right click on it and select **Edit Layout**
4. In this new window click on the **+** then **Control** -> **Scriptable Auto Splitter**
   <p>
     <img width=350 alt="Highlighting the + in the Edit Layout window" src="https://github.com/user-attachments/assets/7b3a5616-ca09-4dc2-a52b-36e23a3e3fd9" />
     <img width=350 alt="Showing where to find the " src="https://github.com/user-attachments/assets/37d0932a-95c4-48da-90e0-5de61d7dbefa" />
   </p>
5. Double click on the **Scriptable Auto Splitter** you added
6. Click **Browse** and select the path to the **Constance.asl** file you downloaded
<img alt="Showing the Layout Settings window" src="https://github.com/user-attachments/assets/d84a59ec-01d4-4f8c-9e6c-540512117f0c" /> <br>
7. Finally click **Ok** in the **Layout Settings** then the **Layout Editor**

> [!IMPORTANT]
> Don't forget to change the display to Game Time (Right click on Livesplit -> **Compare Against** -> **Game Time**)
<img alt="Showing where to Compare against Game Time" src="https://github.com/user-attachments/assets/4c8cc3ab-d582-46e7-97e0-09802d5ceb60" />
